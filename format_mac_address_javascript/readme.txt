MAC Address Formatting

Format an alphanumeric string to a propper MAC address: AB:CD:EF:12:34:56.

1. Change all letters to upper case:
    a1:B2:C3.45:dE-6f => A1:B2:C3.45:DE-6F

2. Remove non-alphanumeric characters:
    A1:B2:C3.45:DE-6F => A1B2C345DE6F

3. Add a colon ':' after every second letter:
    A1B2C345DE6F => A1:B2:C3:45:DE:6F
